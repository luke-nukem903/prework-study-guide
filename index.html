HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The style element controls what CSS styles are associated with a website.
  • The link element connects an external resource to the page.
  • The meta element details metadata such as:
    1. Character Set
    2. Description
    3. Keywords
    4. Author
    5. Viewpoint
    These five items in the meta element help with Search Engine Optimization (SEO).

Git

  • git status: checks what branch we are currently on.
  • git checkout -b "branch-name": creates a new branch and switches to it.
  • git clone [url]: retrieve an entire "repo" (repository) from a hosted location via URL.
  • git branch: Lists all branches of a project. A "*" will appear next to the currently active branch.
  • git commit -m "[descriptive message]": Commit staged content as a new commit snapshot.
  • git add -A: Commits all files in the working branch.

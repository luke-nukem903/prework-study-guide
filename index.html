HTML

  • HTML is the websites building blocks (structure).
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The style element controls what CSS styles are associated with a website.
  • The link element connects an external resource to the page.
  • The meta element details metadata such as:
    • Character Set.
    • Description.
    • Keywords.
    • Author.
    • Viewpoint.
    These five items in the meta element help with Search Engine Optimization (SEO).

CSS

  • CSS (Cascading Style Sheets) defines how a webpage is styled.
    • Colors.
    • Aesthetics.
    • Visual Layout.
    • Orginization.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • Three ways to style a webpage using CSS:
    • Inline CSS.
    • Internal CSS style sheet.
    • External CSS style sheet.

Git

  • git status: checks what branch we are currently on.
  • git checkout -b "branch-name": creates a new branch and switches to it.
  • git clone [url]: retrieve an entire "repo" (repository) from a hosted location via URL.
  • git branch: Lists all branches of a project. A "*" will appear next to the currently active branch.
  • git commit -m "[descriptive message]": Commit staged content as a new commit snapshot.
  • git add -A: Commits all files in the working branch.

JavaScript

  • JavaScript is linked to the HTML within the "body" element.
  • JavaScript is used to give User interactions to a webpage.
  • functions are used to instruct the computer to preform a task in a ceratin order.
  • functions can be called multiple times throught code if a Developer wishes to have the code run at different times.
  • for and if statements are used to run code conditionally.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
